## Abstract

資料視覺化 (Data Visualization) 的一般使用者很少會用一些準則或理論去檢視它,但若對於訊息呈現上的流程與細節深入了解，在設計的速度以及達到溝通目的必然會順暢許多。視覺化背後的準則和原理揭示了很多包含資料與模型、知覺心理學、設計等專業的結合。本講題將從視覺化的語言出發，解析圖表的構成，讓實作者了解心法後能跳脫各種視覺化工具的框架，在應用情境中選擇並設計適合的視覺化流程。
